A word on function keys

The F1–F10 function keys are normally reserved for OS X system functions like Exposé and Dashboard. To make these keys work normally in DOS, hold down fn as you press the function key.

This behaviour can be toggled from the OS X Keyboard preferences pane.

You can also send function key signals directly to DOS using the Emulation ▸ Send Key menu.

Preventing arrow-keys from switching OS X Spaces

The default OS X keyboard shortcuts to switch Spaces are Ctrl ^+⇠⇡⇣⇢. These can interfere with games that use Ctrl ^ as an action button. You can change or disable these shortcuts:

  • In OS X 10.6 and below, via the Exposé and Spaces preferences pane.
  • In OS X 10.7, via the Keyboard preferences pane instead.

Mac keyboards versus PC keyboards

Boxer emulates an MS-DOS PC keyboard layout, and special characters (such as punctuation) are often in different places than where they’re labelled your Mac keyboard.

You may have to try nearby keys and Shift ⇧-ed keys to find the character you’re looking for. On non-US keyboards, you can hold down the right-hand Opt ⌥ key to access additional special characters.

Some keys found on a DOS-era PC keyboard, like Insert and NumLock, don’t exist at all on a Mac keyboard. If you have a game that needs these keys, you can send them using the Edit ▸ Send Key menu.
