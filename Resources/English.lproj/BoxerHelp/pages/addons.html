Installing a game expansion or add-on

Some MS-DOS games received official expansion packs, which usually came on their own diskettes or CD-ROMs and require separate installation. To install an expansion:

  1. Launch the original game while holding down Opt ⌥ to start it up at the DOS prompt.

  3. Add the expansion as a new DOS drive, by doing one of the following:

    • If the expansion is a physical CD: insert the CD into your Mac’s DVD drive.
    • If the expansion is a disc image: double-click on it to mount it as an OS X volume.
    • If the expansion is a folder of install files: drag it into the DOS window.
  4. Browse the expansion’s files in Finder to locate the installer program for the expansion: usually install.exe, setup.exe or similar.

  5. Double-click on the installer program in Finder to launch it in DOS. Follow the installer’s instructions to complete the installation.

    If the installer asks for the location of the original game, enter the path on drive C in which the original game was installed: e.g. “C:\GAMENAME”. If you’re not sure, browse the gamebox in Finder to locate the game’s folder.

  6. If the expansion came on a CD-ROM or disc image, it may require its own CD as a copy protection measure. To ensure the expansion can always find its CD, you should import the CD into the gamebox and familiarize yourself with how to swap CDs while playing.
