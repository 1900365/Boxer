Dealing with CD-ROM copy protection

Many DOS CD-ROM games have copy-protection which checks if the game’s CD is in the drive. Boxer’s CD emulation sometimes isn’t enough to convince the game that the CD is the real thing.

If the game keeps telling you to insert the CD, even when the CD is in your Mac’s DVD drive, then try making a disc image instead. Disc images provide more accurate CD emulation to get around this check.

Making a disc image from a CD-ROM

  1. Open OS X’s Disk Utility.
  2. Select the CD-ROM volume in the left hand sidebar, then click “New Image”.
  3. Change the Image Format to “CD/DVD Master”, then click “Save”.

This will create a new .cdr disc image on your Desktop.

Replacing the old CD-ROM with the new disc image

When importing a game from CD-ROM, Boxer copies the CD into the gamebox as a fake CD-ROM folder. You’ll need to replace this folder with your new disc image:

  1. Browse the gamebox contents in Finder
  2. Find the folder named .cdrom and delete it
  3. Drop your new disc image into the gamebox in its place.

From now on, Boxer will use the disc image as the game CD whenever you launch the gamebox.

Alternatively, you can just reimport the game from the new disc image.
