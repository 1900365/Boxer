Game installation tips and sound card settings

Always install games by dropping them onto the Import window or “Import a new game” droplet.
This will run the game’s installer if necessary, and bundle the game into a new gamebox.

Telling the game where to install

Choose any location on drive C. The path within drive C doesn’t matter, so use whatever the game suggests.

When importing a game, Boxer creates an empty drive C inside a new gamebox.
Installing to drive C will put the game files directly into the gamebox, where they belong.

Choosing an install size

Many CD-ROM games offer a choice of installation sizes, where smaller installations leave more data on the CD.

For most games you should choose the smallest install size, because Boxer will import the whole CD into the gamebox regardless. A larger install size wastes disk space and will have no effect on loading times.

Some games disable features or leave out content at smaller installation sizes.
You should choose the maximum size for those games.

Choosing a sound card

Many games will happily autodetect the correct sound card settings. If you need to choose a sound card manually, then Boxer supports the following:

Sound Blaster/Pro/16

The most reliable choice and supported by almost all games. Choose Sound Blaster 16 if available, otherwise Pro or regular Sound Blaster. Use the following settings if asked:

I/O Address:220
IRQ:7
DMA:1

General MIDI (“MPU-401 compatible”)

This provides high-quality music (better than Sound Blaster) and is supported by most later DOS games. If the game lets you choose separate soundcards for digital audio and music, choose Sound Blaster 16 for digital audio and General MIDI for music.

I/O Address:330

Gravis Ultrasound/Max (“GUS”)

Not widely supported: but in games that properly support it, this can provide more channels of digital audio and better sampled music than Sound Blaster or General MIDI.

If the game’s music card option is labelled “Gravis Ultrasound MIDI synth” then it doesn’t provide native Gravis Ultrasound music, and you should choose General MIDI instead.

I/O Address:240
IRQ:5
DMA:3

AdLib

This music-only card was supported by many older games. You should choose Sound Blaster instead if available.

I/O Address:388
