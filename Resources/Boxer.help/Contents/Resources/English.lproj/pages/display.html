Display and rendering options

Rendering styles

Rendering styles smooth out jagged pixels or apply retro effects to the game graphics. You can select styles from the Display Preferences panel or the View ▸ Rendering Style menu.

To avoid slowdowns, rendering styles are not applied to games in high-resolution (640×480+) modes.

Full Screen mode

Make the game window fill the whole screen by clicking the View ▸ Enter Full Screen menu item or by pressing Cmd ⌘+F.

While in full screen, pressing Cmd ⌘+F will switch back to windowed mode. You can also Cmd ⌘+click in full screen to access the menu bar and the Dock.

If you want to always play games in full screen, enable Start up games in full screen in the Display Preferences panel.

Aspect ratio correction

DOS games were originally designed for display on 4:3 CRT displays, which would stretch the video signal to fill the whole screen. For modes like 320×200, this meant that pixels would be taller rather than square.

By default, Boxer stretches games to their originally-intended ratio. If you prefer sharp square pixels, then disable the Always use 4:3 aspect ratio option in the Display Preferences panel.

You can also toggle this setting with the View ▸ Use 4:3 Aspect Ratio menu item.
