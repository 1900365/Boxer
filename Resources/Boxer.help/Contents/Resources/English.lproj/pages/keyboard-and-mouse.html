Mouse modifiers

These modifiers will help you play DOS games on a trackpad or one-button mouse:

A word on function keys

The F1–F10 function keys are normally reserved for OS X system functions like Exposé and Dashboard. To make these keys work normally in DOS, hold down fn as you press the function key.

This behaviour can be toggled from the OS X Keyboard and Mouse preferences pane.

You can also send function key signals directly to DOS using the Edit ▸ Send Key menu.

Mac keyboards versus PC keyboards

Boxer emulates an MS-DOS PC keyboard layout, and special characters (such as punctuation) are often in different places than they’re labelled your Mac keyboard. You may have to try nearby keys and Shift ⇧-ed keys to find the character you’re looking for.

Some keys found on a DOS-era PC keyboard, like Insert and NumLock, don’t exist at all on a Mac keyboard. If you have a game that needs these keys, you can send them using the Edit ▸ Send Key menu.
