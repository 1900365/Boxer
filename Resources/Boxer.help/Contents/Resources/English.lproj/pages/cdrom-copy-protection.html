Dealing with CD-ROM copy protection

Many DOS CD-ROM games have copy-protection which checks if the game’s CD is in the drive. Boxer’s CD emulation sometimes isn’t enough to convince the game that the CD is the real thing.

If the game keeps telling you to insert the CD, even when the CD is in your Mac’s DVD drive, then try making a disc image instead: these provide more accurate CD emulation to get around this check.

Making a disc image from a CD-ROM

  1. Open OS X’s Disk Utility.
  2. Select the CD-ROM volume in the left hand sidebar, then click “New Image”.
  3. Change the Image Format to “CD/DVD Master”, then click “Save”.

    This will create a new .cdr disc image on your Desktop.

  4. Drag the new disc image onto Boxer’s Import window to reimport the game.
